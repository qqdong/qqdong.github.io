非暴力沟通阅读分享（六）

发表于 2018-09-23 | 分类于 summary

每逢佳节倍思亲，中秋节要到了，老婆和老妈都回老家了，「孤零零」的只剩我一人。放假第一天想着放松一下，就在家追了一个老剧，看的时间长了，头昏脑涨的，颓废的一天就这样过去了。很羡慕那些有自己兴趣爱好的人，一个人也可以很真实，做着有意义的事情。

今天的北京，云彩特别漂亮，加上这个季节的秋高气爽，很适合出去游玩。明天就是中秋节了，祝愿你及你们的家人身体健康，幸福美满。

本篇继续分享非暴力沟通，是最后一部分了，感兴趣的可以回顾下前面的分享。

倾听内心的声音

在成长的过程中，我们学习了许多无益的知识，这些知识来自善意的父母、老师或别的人，由于这些教诲看起来是那样天经地义，也就不再去探讨它们的合理性。同样的，我们在社会文化影响下形成的一些不良积习，渗透了生活的各个方面，以致我们察觉不到它们的存在。只有深入了解我们的生存状态，才能看清它们的危害，并形成新的想法及行为方式，来满足身心健康的需要。

然而，对于大部分人来说，倾听和表达自己的需要并不容易。一般来说，我们的文化倾向于把个人需要看作是消极的、具有破坏性的。

心灵环保

在生活中，有时我们会感到沮丧，是因为一个人处于激烈的内心冲突之中，无所适从。沮丧意味着，我们不了解自己的需要–我们不知道自己到底想要什么以及如何满足愿望。

如果以苛刻的态度对人对己，我们的心情也好不到哪里去。通过运用非暴力沟通，不再试图分析自己或他人有什么毛病，而是用心去了解我们的需要，我们的内心就会逐渐变得平和。

作者举了一个例子：为了能够更好地照顾自己，一位女士报名参加了一个历时三天的非暴力沟通研讨班，在活动期间，她发现自己的生活态度有了明显的改变。

女士回忆说，在第二天清晨醒来时，她头痛得十分厉害。在过去，我的第一反应就是检讨自己到底做错了什么，是否吃了不好的食物？是不是给自己施加了太大的压力？

这一次，在学习了非暴力沟通之后，我问自己的问题是「我现在需要做什么来缓解头痛」。

我坐了起来，做了一些柔和的头部运动，接着，起床到外面走了走，回来后又做了一些别的事情，让自己放松下来。过了一会儿，头没那么痛了。

这时，我想了想昨天在研讨班中的经历，我发现，头痛很可能是因为昨天我不怎么留意身体的状态，现在，身体是通过头痛来提醒我「请留意我的需要」。

于是，在接下来两天的研讨班中，我就用心体会身体的状态，并注意调节和放松。

表达感激

用非暴力沟通的方式表达感激时，我们只是为了庆祝他人的行为提升了我们的生活品质，而不是想得到任何回报。

表达感激的方式包含三个部分：第一，对方做了什么事情使我们的生活得到了改善；第二，我们有哪些需要得到了满足；第三，我们的心情怎么样。在表达感激时，这三部分的先后次序并不重要，用语言具体地描述这三部分，可以确保对方能够明白我们的意思。

作者举了一个例子，是作者和一个非暴力研讨班参加者的一段对话：

参加者：（在研讨班结束后走进我）马歇尔，你真了不起！

作者：我想知道你为什么这么说。

参加者：为什么呢？

作者：在我的一生中，人们用各种各样的话来评价我，这对我并没有多大帮助，我需要更多的信息来理解你的评论。

参加者：什么样的信息？

作者：首先，我想知道，我做了什么，你认为对你的生活有帮助？

参加者：（她想了想，然后拿起笔记本，指着其中的两段话说）就是这两点，你今天在研讨班中提到它们。

作者：哦，你很欣赏这两个观点。听到这两个观点时，你的心情怎么样？

参加者：我感觉一阵轻松，对将来更有信心了。

作者：现在，我想知道，这两个观点使你什么样的需要得到了满足？

参加者：我没法和儿子沟通，他今年18岁了，我一直盼着能有办法和他沟通，你今天的这两个观点给我很大的启发。

接受别人的感激

在听到别人表达对我们的感激时，会觉得别扭。我们不知道自己是否真的那么好。甚至会担心别人对我们有所期待，特别是老师或经理试图通过赞扬来提高我们的学习和工作效率。此外，还可能担心在将来失去别人的赏识。

生活在商品社会中，我们也许不太习惯单纯地付出与接受。然而，非暴力沟通鼓励我们注意倾听别人所表达的感激：我们做了什么有益的事情，他们有哪些需要得到了满足，以及这激发了他们怎样的情感。

我们每个人都有能力提升他人的生活品质。

虽然人们在听到感激时会不太自在，但绝大多数的人渴望得到他人的肯定和感激。对感激的渴望不仅存在于工作场合，而且也存在于家庭中。

作者提到了下面的事情：

有一天晚上，当我向我的儿子布莱特指出他没做好家务活时，他反驳说「爸，你想想，你是不是倾向于指出问题，而很少肯定我」，他的话在我脑海中盘绕了许久，我发现我总是想做得更好，却很少肯定已有的成绩。

我刚刚结束一个有100多人参加的研讨班，除了一个人，其他人都高度评价这次的研讨班，然而，我牢记的却是那个人的不满。

从那以后，我就更加留意生活中积极的一面，并表达我的感激之情。
